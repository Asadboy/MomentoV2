# Quick SwiftLint Installation

The SwiftLint package has been downloaded. To install it, run this command in your terminal:

```bash
sudo installer -pkg /tmp/SwiftLint.pkg -target /
```

Then verify it's installed:

```bash
swiftlint version
```

After installation, you can run:

```bash
cd /Users/asad/Documents/Momento
./lint.sh
```

This will show any linting issues in your code!

---

**Note**: You'll need to enter your Mac password when running the `sudo` command.
